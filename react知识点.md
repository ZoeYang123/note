##### 1.jsx（javascript XML的简写）

JSX是一种javascript的语法扩展，运用于react架构中，你可以将任何的javascript表达式封装在花括号里，然后嵌入到JSX中。

JSX -> React.createElement() -> 虚拟DOM(JS对象) -> 真实DOM
